# Portafolio DevOps de Diego Diaz

Sitio personal construido con React y Vite. Incluye animaciones nativas, diseño responsive,
experiencia profesional, stack técnico, proyecto destacado, formación y enlaces de contacto.

## Ejecutar en local

```bash
npm install
npm run dev
```

## Compilar para producción

```bash
npm run build
```

La carpeta `dist/` resultante puede publicarse en GitHub Pages, Netlify, Vercel o cualquier
hosting estático. `base: "./"` permite desplegar el sitio desde una subcarpeta.
